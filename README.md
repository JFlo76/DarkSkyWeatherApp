# DarkSky API Weather App
## Weather App built upon the [Angular Seed Project](https://github.com/angular/angular-seed) and the [Dark Sky Forecast API](https://developer.forecast.io/)
(currently a work in progress)

# Setup
Clone from git
```
git clone https://github.com/JFlo76/DarkSkyWeatherApp
cd DarkSkyWeatherApp
```

# Install Dependencies
```
npm install
```

# Running the App
```
npm start
```
